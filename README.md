# Twitter-Sentiment-Analysis-NLP
Implementation of a Twitter Sentiment Analysis project using NLP and Random Forest. This repository demonstrates tweet preprocessing, TF-IDF feature extraction, model training, and a basic Streamlit interface for real-time sentiment prediction. Suitable for learning and demonstration purposes.

Features:
- Preprocessing tweets: convert to lowercase, remove mentions and hashtags.
- TF-IDF vectorization for feature extraction.
- Random Forest classifier for sentiment prediction.
- Streamlit-based UI to interactively predict tweet sentiment.
